Help

You can use any conjunction of the following in the top left input field

Parameters

  • 9x9x9 check if the dimension string of a solution (rows x columns x rounds) contains the query string 9x9x9
  • deadbeef check if the sha1 hash of the solution contains the query string deadbeef

Flags

  • best only keep the best known solution for each dimension
  • optimal only keep solutions that are known to be optimal
  • unsat only keep solutions that are unsatisfiable
  • help shows this help text

Any prefix of a flag can be used instead of the flag itself.

Negation

Any query part (except help) can be negated using !, for example

  • !deadbeef check if the sha1 hash of the solution does not contain the query string deadbeef
  • !unsat only keep solutions that are satisfiable
